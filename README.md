## 🛡️ Safety

The script operates entirely client-side and does not modify Roblox server data directly. All currency manipulations are performed via memory injection — the game client sends forged but structurally valid packets to the server. This significantly reduces detection risk compared to server-sided exploits. However, no tool is 100% undetectable. Reasonable use practices: avoid injecting absurd amounts (limit to 50M per session), do not broadcast usage, and never use the tool while a Roblox moderator is visible in-game. The anti-ban engine randomizes memory write intervals to avoid pattern detection. We recommend using the tool on alternate accounts for initial testing.
